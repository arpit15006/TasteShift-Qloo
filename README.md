## 🌟 Project Overview

**TasteShift** is a groundbreaking cultural intelligence platform that creates unprecedented synergy between **Qloo's Taste AI™ API** and **Google's Gemini LLM**. Our solution goes far beyond what either technology could achieve independently, delivering actionable cultural insights that transform how businesses understand and engage with diverse global markets.

### 🎯 The Problem We Solve

In today's globalized world, cultural missteps cost brands millions. **73% of marketing campaigns fail** due to cultural misalignment, and **$62 billion is lost annually** from poor cultural intelligence in product launches.

### 💡 Our Innovation

TasteShift combines Qloo's deep taste intelligence with Gemini's analytical power to create the world's first **AI-powered cultural risk assessment and opportunity identification platform**.

## 🏗 Installation & Setup

### 📋 **Prerequisites**
- Python 3.11+
- Git
- Modern web browser

### ⚡ **Quick Start**
```bash
# Clone the repository
git clone https://github.com/arpit15006/TasteShift-Qloo.git
cd TasteShift-Qloo

# Install dependencies
pip install -r requirements.txt

# Set environment variables (copy from .env.example)
cp .env.example .env

# Initialize database
python init_database.py

# Launch application
python main.py
```
